cyclops.data.aggregate.Aggregator#

class Aggregator(aggfuncs, timestamp_col, time_by, agg_by, timestep_size=None, window_duration=None, window_start_time=None, window_stop_time=None, imputer=None, agg_meta_for=None)[source]#

Bases: TransformerMixin

Equal-spaced aggregation, or binning, of time-series data.

Computing aggregation metadata is expensive and should be done sparingly.

aggfuncs#

Aggregation functions mapped from column to aggregation type. Each value is either function or string, e.g., {col_name: MEAN}. If a function, it should accept a series and return a single value.

Type:

dict

timestamp_col#

Name of the timestamp column in the data provided.

Type:

str

time_by#

Name of columns by which to group to determine the bucket times.

Type:

list of str

agg_by#

Name of columns by which to group to perform aggregation.

Type:

list of str

timestep_size#

Time in hours for a single timestep, or bin.

Type:

float

window_duration#

Time in hours for the aggregation window. If None, the latest timestamp for each time_by group is used as the window stop time.

Type:

float or None

window_start_time#

An optionally provided window start time for each time_by group.

Type:

pd.DataFrame or None

window_stop_time#

An optionally provided window stop time for each time_by group.

Type:

pd.DataFrame or None

agg_meta_for#

Columns for which to compute aggregation metadata.

Type:

list of str or None

window_times#

The start/stop time windows used to aggregate the data.

Type:

pd.DataFrame or None

imputer#

An imputer to perform aggregation.

Type:

AggregatedImputer or None

num_timesteps#

The number of timesteps in the aggregation window.

Type:

int or None

Notes

aggfuncs is a dictionary of aggregation functions mapped from column to aggregation type. Each value is either function or string, e.g., {col_name: MEAN}. If a function, it should accept a series and return a single value. If a string, it should be one of the following:

  • mean

  • median

  • std

  • var

  • min

  • max

  • count

  • sum
